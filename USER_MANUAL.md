## 1. 로그인

### 접속 방법
1. 브라우저에서 어드민 주소 접속
2. 아이디와 비밀번호 입력
3. 로그인 버튼 클릭

### 로그인 화면
| 항목 | 설명 |
|------|------|
| 아이디 | 관리자 계정 아이디 입력 |
| 비밀번호 | 관리자 계정 비밀번호 입력 |

> 로그인 후 자동으로 대시보드로 이동합니다.

## 2. 대시보드

| 대기중 주문 | 처리 대기 중인 주문 수 |

## 3. 상품 관리

### 3.1 상품 목록 보기

왼쪽 메뉴에서 **상품 관리** 클릭

#### 검색 및 필터
| 필터 | 설명 |
|------|------|
| 검색 | 상품명, 설명으로 검색 |
| 카테고리 | 카테고리별 필터링 |
| 메인 노출 | 메인 페이지 노출 여부 (전체/노출/숨김) |

#### 상품 목록 항목
| 항목 | 설명 |
|------|------|
| 상품명 | 상품 이름 (클릭하면 수정 페이지로 이동) |
| 카테고리 | 상품 카테고리 |
| 가격 | 판매 가격 |
| 메인노출 | 메인 페이지 노출 상태 |
| 뱃지 | NEW, SALE 뱃지 표시 여부 |
| 품절상태 | 품절/판매중 상태 |
| 판매기간 | 판매 시작~종료 기간 |
| 관리 | 복제, 수정, 삭제 버튼 |

---

### 3.2 상품 등록하기

1. **+ 상품 등록** 버튼 클릭
2. 아래 정보 입력

#### 기본 정보 (필수)
| 항목 | 설명 | 필수 |
|------|------|:----:|
| 상품명 | 상품 이름 | O |
| 카테고리 | 상품 분류 선택 | O |
| 가격 | 판매 가격 (숫자만) | O |
| 할인률 | 할인 퍼센트 (0~100) | |
| 상품 설명 | 상품 상세 설명 | |

#### 표시 설정
| 항목 | 설명 |
|------|------|
| 메인 페이지에 노출 | 체크 시 메인에 표시 |
| 품절 상태로 설정 | 체크 시 품절 표시 |
| NEW 뱃지 표시 | 신상품 뱃지 표시 |
| SALE 뱃지 표시 | 세일 뱃지 표시 |

#### 이미지 업로드
| 항목 | 설명 |
|------|------|
| 상품 메인 이미지 | 목록/상세에 표시되는 대표 이미지 |
| 상세 설명 이미지 | 상품 상세 페이지에 표시되는 이미지들 |

#### 판매 기간 (선택)
| 항목 | 설명 |
|------|------|
| 판매 시작일시 | 판매 시작 날짜와 시간 |
| 판매 종료일시 | 판매 종료 날짜와 시간 |

> 판매 기간을 설정하지 않으면 상시 판매됩니다.

#### 상품 옵션
상품에 선택 옵션이 있는 경우 (예: 사이즈, 색상) 옵션을 추가할 수 있습니다.

#### 추가상품
함께 판매할 추가 상품을 설정할 수 있습니다.

3. **상품 등록** 버튼 클릭하여 저장

---

### 3.3 상품 수정하기

1. 상품 목록에서 수정할 상품의 **수정** 버튼 클릭
   - 또는 상품명 클릭
2. 내용 수정
3. **변경사항 저장** 버튼 클릭

---

### 3.4 상품 복제하기

기존 상품을 복사해서 새 상품을 만들 때 유용합니다.

1. 복제할 상품의 **복제** 버튼 클릭
2. 확인 창에서 **확인** 클릭
3. 복제된 상품이 생성됨
4. 필요한 부분만 수정하여 저장

---

### 3.5 상품 삭제하기

1. 삭제할 상품의 **삭제** 버튼 클릭
2. 확인 창에서 **삭제** 클릭

> 삭제된 상품은 복구할 수 없습니다.

---

## 4. 주문 관리

### 4.1 주문 목록 보기

왼쪽 메뉴에서 **주문 관리** 클릭

#### 검색 및 필터
| 필터 | 설명 |
|------|------|
| 검색 | 주문번호, 이름, 이메일, 전화번호로 검색 |
| 결제 상태 | 결제완료, 대기중, 취소됨 |
| 상품명 | 특정 상품 주문만 보기 |
| 접수 담당자 | 접수 담당자별 필터 |
| 상담 담당자 | 상담 담당자별 필터 |
| 정렬 | 최신순, 오래된순, 금액순 |
| 기간 | 시작일~종료일 |

**초기화** 버튼: 모든 필터를 초기 상태로 되돌림

---

### 4.2 주문 상태 탭

주문은 상담 상태에 따라 탭으로 구분됩니다.

| 탭 | 설명 |
|------|------|
| 접수 필요 | 새로 들어온 주문, 담당자 배정 필요 |
| 상담 필요 | 담당자 배정됨, 상담 진행 필요 |
| 보류 | 상담 보류 상태 |
| 배송필요(상담완료) | 상담 완료, 배송 처리 대기 |
| 배송보류 | 배송 일시 보류 |
| 배송중 | 배송 진행 중 |
| 배송완료 | 배송 완료 |
| 취소건 | 취소된 주문 |

---

### 4.3 주문 처리 흐름

```
접수 필요
    ↓ (담당자 배정)
상담 필요 ←→ 보류
    ↓ (상담 완료)
배송필요(상담완료) ←→ 배송보류
    ↓ (배송 시작)
배송중
    ↓ (배송 완료)
배송완료
```

---

### 4.4 담당자 배정하기

1. 주문 목록에서 해당 주문 찾기
2. **접수 담당자** 드롭다운에서 담당자 선택
3. **상담 담당자** 드롭다운에서 담당자 선택

> 담당자를 선택하면 자동 저장됩니다.

---

### 4.5 주문 상태 변경하기

#### 개별 변경
1. 주문을 클릭하여 상세 페이지 이동
2. **상담 상태** 영역의 버튼으로 상태 변경

#### 일괄 변경
1. 변경할 주문들 체크박스 선택
2. 하단에 나타나는 버튼 클릭
   - 예: "상담완료로 이동"

---

### 4.6 배송 엑셀 추출하기

**배송필요(상담완료)** 탭에서 배송 정보를 엑셀로 추출할 수 있습니다.

1. 배송필요(상담완료) 탭으로 이동
2. 추출할 주문들 체크박스 선택
3. **엑셀 추출** 버튼 클릭
4. 엑셀 파일 다운로드됨
5. 선택한 주문들이 자동으로 **배송중**으로 변경됨

> 파일명: `배송목록_YYYY-MM-DD.xlsx`

---

### 4.7 주문 상세 보기

주문 목록에서 주문을 클릭하면 상세 페이지로 이동합니다.

#### 유저 정보
| 항목 | 설명 |
|------|------|
| 주문자 | 주문자 이름 (클릭하면 복사) |
| 연락처 | 전화번호 (클릭하면 복사) |
| 이메일 | 이메일 주소 (클릭하면 복사) |

#### 주문 상태
| 항목 | 설명 |
|------|------|
| 초진/재진 여부 | 첫 방문/재방문 구분 |
| 상담 상태 | 현재 상담 진행 상태 |
| 결제 상태 | 결제 완료/대기/취소 |
| 주문일시 | 주문한 날짜와 시간 |

#### 문진 결과
고객이 작성한 건강 상담 정보가 표시됩니다.
- **전체 복사** 버튼으로 모든 정보를 한 번에 복사할 수 있습니다.
- 각 항목을 클릭하면 해당 항목만 복사됩니다.

#### 주문 정보
| 항목 | 설명 |
|------|------|
| 주문번호 | 고유 주문 번호 |
| 상품 금액 | 주문 상품 합계 |
| 배송비 | 배송 비용 |
| 쿠폰 할인 | 쿠폰 적용 금액 |
| 포인트 사용 | 사용한 포인트 |
| 최종 결제 금액 | 실제 결제 금액 |

#### 배송지 정보
| 항목 | 설명 |
|------|------|
| 받는 분 | 수령인 이름 |
| 연락처 | 수령인 전화번호 |
| 주소 | 배송 주소 |

#### 관리자 메모
주문에 대한 메모를 작성할 수 있습니다.
1. 메모 입력
2. **메모 저장** 버튼 클릭

---

### 4.8 상품 옵션 설정하기

옵션이 있는 상품의 경우, 상담 완료 전에 옵션 설정이 필요합니다.

1. 주문 상세에서 **설정하기** 또는 **설정 변경** 버튼 클릭
2. 옵션 선택
3. 저장

> 옵션 설정이 완료되지 않은 주문은 "상담완료"로 이동할 수 없습니다.

---

### 4.9 주문 취소하기

1. 주문 상세 페이지에서 **주문 취소** 버튼 클릭
2. 취소 사유 입력
3. **결제 취소** 버튼 클릭

> 결제가 즉시 취소되며, 재고와 혜택이 모두 회수됩니다.

---

### 5.5 포인트 내역 보기

1. 유저 상세 페이지에서 **포인트 내역 보기** 버튼 클릭
2. 적립/사용 내역 확인

---

### 5.6 유저에게 쿠폰 발급하기

3. **발급** 클릭

## 6. 쿠폰 관리

### 6.1 쿠폰 목록 보기

왼쪽 메뉴에서 **쿠폰 관리** 클릭

#### 검색 및 필터
| 필터 | 설명 |
|------|------|
| 검색 | 쿠폰 코드, 이름, 설명으로 검색 |
| 상태 | 전체, 활성, 비활성 |

#### 쿠폰 목록 항목
| 항목 | 설명 |
|------|------|
| 코드 | 쿠폰 코드 (복사 버튼 있음) |
| 쿠폰명 | 쿠폰 이름과 설명 |
| 할인 | 할인율(%) 또는 할인금액(원) |
| 적용 상품 | 모든 상품 또는 특정 상품 |
| 최소 구매금액 | 쿠폰 사용 가능 최소 금액 |
| 유효기간 | 쿠폰 만료일 |
| 상태 | 활성/비활성 (클릭하여 변경 가능) |
| 관리 | 발급, 수정, 삭제 버튼 |

---

### 6.2 쿠폰 생성하기

1. **+ 쿠폰 생성** 버튼 클릭
2. 쿠폰 정보 입력

| 항목 | 설명 | 필수 |
|------|------|:----:|
| 쿠폰 코드 | 고유한 쿠폰 코드 | O |
| 쿠폰명 | 쿠폰 이름 | O |
| 설명 | 쿠폰 설명 | |
| 할인 유형 | 퍼센트(%) 또는 금액(원) | O |
| 할인 값 | 할인율 또는 할인금액 | O |
| 최소 구매금액 | 쿠폰 사용 최소 금액 | |
| 적용 상품 | 전체 또는 특정 상품 선택 | |
| 유효기간 | 쿠폰 만료일 | O |

3. **저장** 클릭

---

### 6.3 쿠폰 수정하기

1. 수정할 쿠폰의 **수정** 버튼 클릭
2. 정보 수정
3. **저장** 클릭

---

### 6.4 쿠폰 활성/비활성

쿠폰 목록에서 **상태** 뱃지를 클릭하면 활성/비활성이 전환됩니다.

- **활성**: 쿠폰 사용 가능
- **비활성**: 쿠폰 사용 불가

---

### 6.5 쿠폰 발급하기

특정 유저에게 쿠폰을 발급합니다.

1. 발급할 쿠폰의 **발급** 버튼 클릭
2. 유저 검색 (이메일, 이름, 전화번호)
3. 발급할 유저 선택
4. **발급** 클릭

---

### 6.6 쿠폰 삭제하기

1. 삭제할 쿠폰의 **삭제** 버튼 클릭
2. 확인 창에서 **삭제** 클릭

---

## 자주 묻는 질문

### Q: 로그인이 안 돼요
- 아이디와 비밀번호를 정확히 입력했는지 확인하세요.
- Caps Lock이 켜져 있지 않은지 확인하세요.
- 문제가 계속되면 마스터 관리자에게 비밀번호 재설정을 요청하세요.

### Q: 메뉴가 보이지 않아요
- 일반 관리자는 주문 관리만 접근할 수 있습니다.
- 다른 메뉴 접근이 필요하면 마스터 관리자에게 권한 변경을 요청하세요.

### Q: 주문을 상담완료로 이동할 수 없어요
- 옵션 상품의 옵션 설정이 완료되었는지 확인하세요.
- "설정 필요"라고 표시된 상품의 옵션을 먼저 설정해주세요.

### Q: 엑셀 파일이 다운로드되지 않아요
- 브라우저의 팝업 차단이 해제되어 있는지 확인하세요.
- 다른 브라우저로 시도해보세요.

### Q: 삭제한 데이터를 복구할 수 있나요?
- 삭제된 상품, 주문, 관리자 등은 복구할 수 없습니다.
- 삭제 전 반드시 확인하세요.
